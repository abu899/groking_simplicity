# Part 1

## Ch 3

액션, 계산 데이터의 차이를 알아보기

### 액션

실행 시점이나 횟수 또는 둘다에 의존
- 부수 효과(side effect), 부수 효과가 있는 함수, 순수하지 않은 함수
- ex) 이메일 보내기, DB 읽기

액션 안에는 계산과 데이터, 또 다른 액션이 숨어 있을 수 있다. 따라서 이를 더 작은 액션과 계산, 데이터로 나눠야하는데 언제 멈춰야하는지를 아는 것이 중요하다.

### 계산

입력으로 출력을 계산
- 순수 함수, 수학 함수
- ex) 최댓값 찾기, 이메일 주소 검증

### 데이터

이벤트에 대한 사실
- 데이터를 쉽게 해석할 수 있도록 표현하는 것이 중요한 기술
